📘 Simple Dictionary App
📌 Project Idea

This project is a simple Dictionary App developed using Android Studio.

The main idea of the app is to allow the user to add words with their meanings, display them in a list, and also edit or delete any word.

✨ Features
Add a new word with its meaning
Display words using RecyclerView
Edit existing words
Delete words by long click
Show simple Toast messages
Save data during screen rotation
📱 How the App Works

When the app is opened, a list of words is displayed.

The user can press the (+) button to add a new word.

By clicking on a word, the user can edit it.

By long pressing on a word, it will be deleted.
